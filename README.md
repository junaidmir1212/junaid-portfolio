# Muhammad Junaid Mir Portfolio

A premium portfolio website built with Next.js and Framer Motion, designed for deployment on GitHub Pages or Vercel.

## Features
- Responsive premium UI
- Smooth animations with Framer Motion
- Downloadable CV button
- Dedicated project case-study section
- Contact form
- Mobile-friendly layout

## Run locally
```bash
npm install
npm run dev
```

## Build for GitHub Pages
```bash
npm install
npm run build
```

The static site output will be generated for deployment.

## Before deployment
- Put your CV PDF in `public/Muhammad_Junaid_Mir_CV.pdf`
- If using the contact form, update the `_next` hidden input in `app/page.tsx` to your final domain if needed
